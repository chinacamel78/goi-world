第二章：欢喜冤家
  • 锦瑟被长老会指定为"萧鲤的引导师兄"——负责帮助萧鲤适应宗门生活。
  • 第一次冲突：萧鲤写代码不写注释。锦瑟要求她补上，萧鲤说："我的代码像诗，诗不需要注释。"锦瑟说："诗不需要注释，但代码需要。因为三个月后，诗还是诗，但代码可能变成谜语。"
  • 萧鲤赌气在传经堂中当众讲了一首诗——用代码写的诗：
  for (int i = 0; i < 100; i++) {
      cout << "I am a fish, swimming in the code." << endl;
  }
全场鼓掌。但锦瑟站起来指出："这个循环会输出100行，但每行都是重复的。如果代码是诗，这首诗的'韵脚'太单一了。" 萧鲤第一次意识到：锦瑟的"死板"背后，是另一种"审美"。
